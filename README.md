Go To Hell
==========

A rage-y way of killing yo processes.

Golang port of [robotlolita's fuck-you](https://github.com/robotlolita/fuck-you)


## Installation

    go get github.com/mattn/go-to-hell/go


## Usage

    go to hell process_name

Kills `process_name`.
